# Harry-Porter
This is an android app built entirely on Kotlin that displays a list of Harry Porter's characters together with the characteristics
Concepts used are :
 1.Retrofit
 2.OkHttp 
 3.LiveData
 4.ViewModel
 5.MVVM
 6.RecyclerView with DiffUtil
 7.Unit Testing
 8.Navigation component
